# Gestion Locative

## Description
Application pour gérer des biens immobiliers.

## Technologies
- React
- Spring Boot
- MySQL

## Installation
1. Installer Java 17
2. Installer Node.js
3. Installer MySQL

## Lancement
1. Lancer le backend : mvnw spring-boot:run
2. Lancer le frontend : npm start
